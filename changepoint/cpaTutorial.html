Changepoint Analysis¶

This is a step-by-step guide to using the changepoint analysis python tool cpa for ONC data. The code can be found on GitHub.

In [11]:
%matplotlib inline

import cpa
import pandas as pd
import numpy as np

Prepare data¶

The cpa module takes data in the form of a numpy array. ONC data can be downloaded as .txt files for easy python handling. Some ferry data is minute-averaged and formatted into a numpy array here. NOTE: data can be prepared any way the user likes, so long as the end results is a numpy array.

In [7]:
# Read in txt file and convert data to numpy array of float values
cdom = pd.read_csv('../bcferriesdata/CDOM/CDOM_jan_mar.txt', sep=';', header=33)
cdom_series = cdom['CDOM Fluorescence [ppb]:DOUBLE'].values
cvals = np.asarray([float(i) for i in cdom_series])

# Minute average the data
cvals = pd.DataFrame(cvals)
cvals.rolling(window=6, center=False).mean()
cvals = np.asarray(cvals)

# Remove nans from CDOM values
ma_cvals = np.ma.masked_invalid(cvals)
cvals_nonan = ma_cvals.compressed()

Analysis¶

The cpa module contains the changepoint class, an instance of which will contain the analysis

In [12]:
# A changepoint object is generated using the prepared data above with bandpass filtering turned on
cpa1 = cpa.changepoint(data=cvals_nonan, filter_on=True)
